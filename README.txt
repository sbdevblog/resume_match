# AI Resume Matcher & Job Fit Analyzer

## About
A Machine Learning project that compares resumes and job descriptions to calculate match percentage and identify missing skills.

## Features
- Resume text extraction (PDF)
- TF-IDF similarity scoring
- Missing & extra skills detection
- Streamlit-based web UI

## Tech Stack
Python | NLP | Scikit-learn | Streamlit | SpaCy

## Run Locally
```bash
git clone https://github.com/YOURUSERNAME/ai-resume-matcher.git
cd ai-resume-matcher
pip install -r requirements.txt
streamlit run src/app.py
